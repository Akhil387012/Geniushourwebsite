What is html?

Html is the coding language used for every website or webpage. It stands for Hypertext Markup Language, and it tells how the text is supposed to be organized, what text to put, what pictures to put, and if you also use CSS(Another language for webpages/websites), what colors to put. Putting links is something html can do too. Html is based on small text codes that are called tags. It uses brackets that are pointed, which you probably think of as "greater then" and "less then" signs. Some examples of websites that use html are Yahoo Finance, Google, Scratch, and a whole lot more!

What is a Text Editor?

A text editor is a computer program that let's people create and edit text files. They don't have to be for coding though. You can use it to type essays or take notes too. In this case, we used the text editor for coding a website. Some examples are Notepad, Notepad++, Atom, and more! There are two types of text, plain text and rich text. Plain text is not formatted, and it is usually what you use if your just putting notes, while rich text is formatted, and has options like "bolded words, italics, colors, underlining, and diverse text structures." Html is plain text. So if you were making a webpage, you would change the format to plain text. This is mostly a problem if you use TextEdit on a Mac computer.

What is CSS?

CSS stands for Cascading Style Sheets, and it is the coding language used to enhance a website. If Html was the building blocks of a house, CSS would be the paint, furniture, and materials. In order to code CSS, you need a separate file called a spreadsheet. You're probably seeing colors in here right now. If you are, that is all from CSS. CSS can also be used for another website coding language called XML and much more. One fact about CSS is that CSS was actually invented by the World Wide Web Consortium after Html was made to solve the problem of large development of websites taking a lot of time to make color for and being really expensive.
